Assignment 1

The initial assignment in our web page authoring course was a foundational exercise in understanding and applying the basics of web development. The primary objective was to familiarize ourselves with the fundamental components of web design, including the configuration of web accounts, the creation of directories, setting appropriate file permissions, and the process of uploading files to a server. This task required us to step away from the convenience of modern web authoring tools such as DreamWeaver, directing us instead to rely on more basic text editors like NotePad++ or TextWrangler. The rationale behind this approach was to immerse us in the nuts and bolts of web development, ensuring that we gained a hands-on understanding of the underlying processes.

This experience was enlightening, offering a deep dive into the intricacies of web page creation. I learned about the various tags in HTML and how they form the skeleton of a web page. Understanding the structure of HTML was crucial, as it serves as the foundation upon which websites are built. Additionally, the assignment introduced me to the interaction between HTML and CSS, highlighting how CSS enhances the appearance of a web page. This practical exercise helped me appreciate the power of CSS in establishing the visual aesthetics of a site. Overall, the assignment was a valuable exercise in understanding the basics of web page authoring, laying a solid foundation for more advanced topics in web development.

  • Mastering Web Account Configuration: Achieve proficiency in setting up and managing your web hosting account, ensuring effective access and control over your server space.
  • Skills in Folder Creation and Management: Develop the ability to create and organize directories on your server, reflecting the structure on your local machine for optimal organization.
  • Expertise in Setting Permissions: Acquire the knowledge to set the right permissions for your files and folders, balancing accessibility and security on the web server.
  • Proficiency in Manual File Authoring: Enhance your understanding of web development by manually coding web pages using text editors like NotePad++ or TextWrangler, focusing on HTML and CSS.
  • Comprehensive Understanding of HTML: Gain in-depth knowledge of HTML, learning about various tags and how to structure your web content effectively.
  • Integration of CSS with HTML: Learn to aesthetically enhance your web pages by integrating CSS with HTML, understanding how to link and apply CSS styles effectively.
  • Efficient Uploading of Files to Server: Master the use of Secure File Transfer Protocol (SFTP) tools like FileZilla to upload your files, making your web page accessible online.
  • Effective Server Interaction: Develop skills in using Secure Shell (SSH) clients like PuTTY for direct server interaction, including basic Linux commands for file management.
